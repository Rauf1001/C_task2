В каждом решении должна быть иерархия классов (как минимум одна) и полиморфизм.

- Управление рабочими строительной компании  
  **Описание**: Компания занимается строительством зданий. У каждого объекта строительства есть этажность и площадь. Рабочие подразделяются на профессии (строители, инженеры, архитекторы). Каждый объект имеет фазы строительства, требующие различных специальностей с разным коэффициентом человеко-часа на м2 трудозатрат.  
  **Функциональность**:
- Создание объектов строительства и управление их состояниями.
- Найм рабочих разных профессий.
- Назначение рабочих на объекты.
- Моделирование процесса строительства пошаговое, один шаг – неделя.
- Учесть, что у компании может быть несколько объектов одновременно
- Строительство может быть приостановлено
- Подсчет (статистика) затраченных ресурсов на строительство, наблюдаем за расходом денег на рабочих
